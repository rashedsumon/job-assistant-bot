# Workflow-Driven AI Job Application Assistant

A modular Streamlit application to analyze job descriptions, tailor resumes, track statuses, and prepare for interviews using a structured, repeatable workflow.

## Local Setup
1. Clone the repository.
2. Install dependencies: `pip install -r requirements.txt`
3. Run the application: `streamlit run streamlit_app.py`

## Deployment on Streamlit Cloud
1. Push this repository to GitHub.
2. Connect your GitHub account to Streamlit Cloud.
3. Deploy the application pointing to `streamlit_app.py` as the main file path.
